# heart-disease-dashboard-using-airtable-ai
# ❤️ Heart Disease Prediction Dashboard (Airtable)

An interactive **Heart Disease Data Analytics Dashboard** built using **Airtable** to manage, explore, and analyze patient health records.  
The dashboard allows users to visualize key health metrics and identify patterns related to heart disease.

This project demonstrates **data analysis, dashboard design, and data visualization** using a no-code analytics platform.

## 📌 Project Overview

The **Heart Disease Prediction Dashboard** helps analyze patient data such as age, blood pressure, cholesterol levels, and chest pain types.

Using Airtable's **Interface Designer, Filters, and Charts**, the dashboard enables users to explore patient records and identify potential risk factors for heart disease.

---

## 🚀 Features

- 📊 Interactive patient records dashboard
- 🔎 Dynamic filters for data exploration
- 📈 Multiple chart visualizations
- 📂 Organized database structure
- 🤖 Airtable AI insights for data analysis

---

## 🗂 Database Structure

### Table: Patient Records

| Field Name | Description |
|------------|-------------|
| Age | Patient age |
| Sex | Male / Female |
| Chest Pain Type | Type of chest pain (1–4) |
| BP | Blood Pressure |
| Cholesterol | Cholesterol level |
| FBS over 120 | Fasting Blood Sugar indicator |
| EKG Results | Electrocardiographic results |
| Heart Disease Status | Presence of heart disease |

## 📊 Dashboard Charts

The dashboard includes the following visualizations:

1. **Line Chart**
   - Monthly Patient Intake Trend

3. **Bar Chart**
   - Heart Disease by Age Group

   - Age vs Cholesterol Relationship

   - Heart Disease by Sex

## 🔎 Filters Available

Users can filter patient data by:

- Heart Disease Status
- Sex
- Chest Pain Type

These filters help analyze trends and identify patterns in the dataset.
